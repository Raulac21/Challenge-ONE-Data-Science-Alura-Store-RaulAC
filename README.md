#  Alura Store Latam – Data Analysis & Visualizations

Este repositorio contiene un análisis exploratorio y visual de datos correspondiente a cuatro tiendas de **Alura Store Latam**, empleando herramientas como **Pandas**, **Matplotlib**.

## Cómo descargar o clonar este proyecto

1. Accede al repositorio en GitHub.  
2. Haz clic en **Code → Download ZIP**.  
3. Extrae el archivo en tu computadora.  
4. Abre el archivo `.ipynb` en **Google Colab**.



---

##  1. Contenido del análisis

###  Carga y preparación de datos
- Lectura de archivos CSV directamente desde GitHub.  
- Normalización y limpieza de nombres de columnas.  
- Cálculo de ingresos totales por tienda.  
- Obtención de datos globales como ventas, calificaciones y costos de envío.

###  Datos por tienda
- Ingreso total  
- Categoría más vendida  
- Promedio de calificación  
- Producto más vendido  
- Costo de envío promedio  

Toda esta información se consolida en la **`TABLA FINAL DE DATOS GENERALES DE TIENDAS`**.
